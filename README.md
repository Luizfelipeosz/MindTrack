# 🧠 MindTrack

Sistema SPA desenvolvido em React + Vite que simula uma plataforma de monitoramento emocional e produtividade.  
O projeto possui autenticação simulada, dashboard analítico, calendário interativo, gráficos e arquitetura modular focada em escalabilidade.

## 🚀 Funcionalidades

- 🔐 Autenticação simulada
- 📊 Dashboard com métricas e gráficos
- 📅 Calendário de atividades
- 🧠 Monitoramento emocional
- 📈 Histórico de desempenho
- 🌙 Dark mode
- 📱 Layout responsivo
- ⚡ Navegação SPA com React Router
- 🧩 Componentização reutilizável

---

## 🛠️ Tecnologias utilizadas

- React.js
- TypeScript
- Vite
- React Router DOM
- Context API
- Recharts
- TailwindCSS

⚙️ Instalação

Clone o projeto:

git clone <repositorio>

Acesse a pasta:

cd mindtrack

Instale as dependências:

npm install

Inicie o projeto:

npm run dev
